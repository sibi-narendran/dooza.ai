# Dooza Brand Guidelines

## Blog Image Generation Prompt Template

All blog hero images must be **1200x630px** and use this consistent style:

```
A soft watercolor and ink illustration of [DESCRIBE YOUR SUBJECT HERE]. The art style is professional, hand-drawn, and clean. The color palette is strictly muted sage green, teal, slate grey, and white on a textured cream paper background. The illustration includes loose sketch lines, soft watercolor washes. Friendly, corporate-tech aesthetic, minimalist composition, high quality.
```

### Image Specs
- **Dimensions:** 1200x630px (16:9 OG image standard)
- **Format:** PNG
- **Location:** `/public/blog/`
- **Naming:** Must match blog slug (e.g., `what-is-openclaw.png`)
